# A*_Sudoku_Solver
Solving sudoku using A* algorithm.

Input file should be A CSV file like this:

9,0,0,0,7,0,0,2,0
0,8,0,0,0,0,0,3,0
6,0,0,0,0,4,7,0,8
0,0,0,0,0,0,0,8,0
0,6,0,0,3,0,9,0,1
4,0,0,5,0,0,0,0,0
0,0,9,0,0,3,0,0,0
0,1,0,0,9,0,6,0,7
0,0,0,0,0,0,2,0,0

Where zeros represent empty cells in sudoku table.
